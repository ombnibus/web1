What is World Wide Web?

World Wide Web (WEB)The World Wide Web, commonly known as the Web, is an information system where documents and other web resources
are identified by Uniform Resource Locators, which may be interlinked by hypertext,
and are accessible over the Internet.
Web browsers receive HTML documents from a web server or a web page from loval storage and render them into multimedia web pages.
